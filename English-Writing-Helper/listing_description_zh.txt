**項目名稱：**
English Writing Helper

**項目摘要：**
將文字翻譯並潤飾成自然英文，可選正式、口語或自訂寫作風格。

**主要說明：**

English Writing Helper 協助非英語母語者在網頁上將文字翻譯並潤飾成自然英文。你可以使用自己的 Gemini 或 OpenAI API Key，選擇適合情境的語氣，並在撰寫 Email、貼文、訊息、筆記時即時取得英文寫法。

**主要功能：**

*   **即時翻譯：** 輸入時或選取文字後，可將非英文內容翻譯成英文。
*   **風格選擇：** 可選擇正式或口語化的英語風格，滿足您的不同需求。
*   **自訂翻譯指令：** 進階使用者可自訂翻譯 prompt，指定語氣、格式或使用情境。
*   **Gemini/OpenAI 強力支援：** 運用 Google Gemini API 或 OpenAI 提供高品質的寫作建議。
*   **個人化設定：** 可使用您自己的 Gemini 或 OpenAI API 金鑰，並設定預設的寫作風格（正式/口語）。
*   **多語介面：** 介面可自動跟隨 Chrome/系統語言，也能在設定中手動切換。
*   **回饋與許願：** 可直接從插件送出問題回報、使用心得或新功能願望。
*   **靈活顯示：** 可在側邊欄或輸入欄位正下方查看翻譯建議。
*   **輕鬆開關：** 根據需求快速開啟或關閉擴充功能。

**為何安裝 English Writing Helper？**

*   **提升自信與效率：** 更自信且快速地進行英語寫作。
*   **學習與適應：** 學習如何用自然的英語語氣表達您的想法。
*   **多功能用途：** 非常適合撰寫電子郵件、社群媒體貼文、訊息等。
*   **無縫整合：** 在幾乎任何網頁的輸入欄位中即時獲得英語建議。

**重要事項：**

*   **需要 API 金鑰：** 此擴充功能需要您自備 Google Gemini 或 OpenAI API 金鑰才能運作。
*   **潛在 API 費用：** 使用所選 AI API 可能會根據您的用量產生費用。詳情請參閱該服務商的定價資訊。
*   **隱私友善用量回報：** 可傳送匿名用量統計協助改善產品。插件只統計翻譯次數與字數，不會傳送原文、譯文、API Key、目前網址或瀏覽紀錄。
*   **目前限制：** 最適用於標準 HTML 輸入欄位和文字區域。對部分富文本編輯器的支援可能有限。
*   **焦點功能：** 協助非英語母語者翻譯並練習英文寫作。
