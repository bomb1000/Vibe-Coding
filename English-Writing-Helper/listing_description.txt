**Item Title:**
English Writing Helper

**Item Summary:**
Translate and polish text into natural English with formal, casual, or custom writing styles.

**Main Description:**

English Writing Helper helps non-native English writers translate and polish text into natural English across the web. Use your own Gemini or OpenAI API key, or use optional Managed Credits if you do not want to set up an API key. Choose the tone that fits your context and keep the helper available while writing emails, posts, messages, notes, and more.

**Key Features:**

*   **Real-time Translation:** Translate non-English text into English as you type or from selected text.
*   **Style Options:** Choose between Formal and Conversational English to suit your needs.
*   **Custom Translation Instructions:** Advanced users can define their own translation prompt for specific tones, formats, or contexts.
*   **Gemini/OpenAI Powered:** Use Google Gemini or OpenAI for high-quality writing suggestions.
*   **Two Usage Modes:** Use your own Gemini/OpenAI API key, or use optional prepaid Managed Credits.
*   **Customizable:** Save your provider, model, writing style, and custom prompt.
*   **Localized Interface:** The interface can follow your Chrome/system language automatically, with manual language switching available in settings.
*   **Feedback & Wishlist:** Send product feedback, bug reports, or feature wishes directly from the extension.
*   **Flexible Display:** View suggestions in a convenient sidebar or directly below your input field.
*   **Easy Control:** Quickly toggle the extension on or off as needed.

**Why Install English Writing Helper?**

*   **Boost Confidence & Efficiency:** Write in English with greater assurance and speed.
*   **Learn & Adapt:** Understand how to express your ideas naturally in different English tones.
*   **Versatile Use:** Ideal for crafting emails, social media posts, messages, and more.
*   **Seamless Integration:** Get instant English suggestions directly on almost any webpage input field.

**Important Notes:**

*   **API Key or Managed Credits:** You can use your own Gemini/OpenAI API key, or use optional prepaid Managed Credits when available.
*   **Potential Costs:** Your own API key may incur provider charges. Managed Credits are prepaid and usage-based.
*   **Managed Credits Privacy:** In Managed Credits mode, selected text is sent to the English Writing Helper backend and AI provider only to complete translation and deduct credits. Source text and translations are not stored in the usage dashboard.
*   **Privacy-Friendly Usage Reporting:** Anonymous usage statistics are requested with a clear first-run choice. The extension counts translation attempts, character totals, provider, model, writing style, and version, but does not send source text, translated text, API keys, page URLs, or browsing history. Managed Credits purchases receive 5% bonus credits while anonymous statistics are enabled.
*   **Current Limitations:** Works best with standard HTML input fields and textareas. Support for some rich-text editors may be limited.
*   **Focus:** Designed for non-native English writers who want to translate and practice writing in English.
