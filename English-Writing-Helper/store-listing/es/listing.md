# Spanish - es

## Title
English Writing Helper

## Summary
Traduce y mejora textos en inglés con estilos formales, conversacionales o personalizados.

## Description
English Writing Helper ayuda a personas que no son hablantes nativas de inglés a traducir y mejorar textos en inglés natural desde cualquier sitio web. Puedes usar tu propia API Key de Gemini u OpenAI, elegir el tono que mejor encaje con tu contexto y mantener el asistente disponible mientras escribes emails, publicaciones, mensajes, notas y más.

Funciones principales

- Traducción en tiempo real: Traduce texto no escrito en inglés desde una selección o mientras escribes.
- Opciones de estilo: Elige entre redacción formal e inglés conversacional natural.
- Instrucciones personalizadas: Define tu propio prompt para tonos, formatos, audiencias o reglas de escritura específicas.
- Compatibilidad con Gemini/OpenAI: Usa Google Gemini u OpenAI para sugerencias de escritura en inglés.
- Ajustes personales: Guarda tu API Key, proveedor, modelo, estilo de escritura y prompt personalizado.
- Interfaz localizada: Sigue automáticamente el idioma de Chrome/sistema o cambia el idioma manualmente.
- Comentarios y deseos: Envía errores, comentarios o solicitudes de funciones directamente desde la extensión.
- Vista flexible: Lee las sugerencias en una barra lateral cómoda mientras sigues trabajando.
- Control sencillo: Activa o desactiva la extensión cuando lo necesites.

¿Por qué instalar English Writing Helper?

Está diseñada para personas que ya saben qué quieren decir, pero necesitan un inglés más claro y natural. El flujo es simple: selecciona texto, traduce, ajusta el estilo, copia el resultado y sigue escribiendo.

Notas de privacidad

Tu API Key se guarda en el almacenamiento de la extensión de Chrome. La extensión no envía automáticamente al desarrollador la URL actual, el historial de navegación, la API Key, el texto original ni la traducción. Las estadísticas anónimas opcionales solo incluyen datos agregados como intentos de traducción, cantidad de caracteres, proveedor, estilo y versión de la extensión.
