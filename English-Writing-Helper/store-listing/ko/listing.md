# Korean - ko

## Title
English Writing Helper

## Summary
글을 자연스러운 영어로 번역하고 다듬으세요. 격식체, 대화체, 사용자 지정 스타일을 지원합니다.

## Description
English Writing Helper는 영어가 모국어가 아닌 사용자를 위한 영어 글쓰기 도우미입니다. 웹 어디에서나 문장을 자연스러운 영어로 번역하고, 상황에 맞게 격식 있는 글쓰기, 자연스러운 대화체, 또는 사용자 지정 prompt를 선택할 수 있습니다. 이메일, 게시물, 채팅, 메모, 문서 등 다양한 영어 작성에 사용할 수 있습니다.

주요 기능

- 실시간 번역: 선택한 문장이나 입력 중인 문장을 영어로 번역합니다.
- 스타일 선택: 격식 있는 영어와 자연스러운 대화체를 빠르게 선택합니다.
- 사용자 지정 번역 지시: 톤, 형식, 대상 독자, 특별한 작성 규칙을 prompt로 지정합니다.
- Gemini/OpenAI 지원: 자신의 Google Gemini 또는 OpenAI API Key를 사용할 수 있습니다.
- 개인 설정: API Key, AI 제공업체, 모델, 글쓰기 스타일, 사용자 지정 prompt를 저장합니다.
- 다국어 인터페이스: Chrome/시스템 언어를 자동으로 따르거나 설정에서 직접 바꿀 수 있습니다.
- 피드백 / 위시리스트: 문제 제보, 사용 경험, 원하는 기능을 확장 프로그램에서 바로 보낼 수 있습니다.
- 편리한 사이드바: 작업 중인 페이지를 떠나지 않고 번역 결과를 확인합니다.
- 쉬운 제어: 필요할 때만 확장 프로그램을 켜거나 끌 수 있습니다.

English Writing Helper를 설치해야 하는 이유

말하고 싶은 내용은 있지만 더 자연스럽고 명확한 영어가 필요한 사람을 위해 만들었습니다. 텍스트를 선택하고, 번역하고, 스타일을 조정하고, 결과를 복사한 뒤 계속 작성할 수 있습니다.

개인정보 안내

API Key는 Chrome 확장 프로그램 저장소에 저장됩니다. 현재 URL, 방문 기록, API Key, 원문, 번역문은 개발자에게 자동 전송되지 않습니다. 선택 사항인 익명 사용 통계에는 번역 횟수, 글자 수, 제공업체, 스타일, 확장 프로그램 버전 같은 집계 정보만 포함됩니다.
