# Japanese - ja

## Title
English Writing Helper

## Summary
文章を自然な英語に翻訳・改善。フォーマル、カジュアル、カスタム文体に対応。

## Description
English Writing Helper は、英語を母語としない人のための英文ライティング補助ツールです。ウェブ上の文章を自然な英語に翻訳し、場面に合わせてフォーマル、カジュアル、またはカスタム prompt による文体を選べます。メール、投稿、チャット、メモ、ドキュメントなど、日常の英文作成に使えます。

主な機能

- リアルタイム翻訳: 選択した文章や入力中の文章を英語に変換。
- 文体の切り替え: フォーマルな文章と自然な会話調を選択。
- カスタム翻訳指示: トーン、形式、読者、特別な書き方のルールを prompt で指定。
- Gemini/OpenAI 対応: 自分の Google Gemini または OpenAI API Key を使用可能。
- 2 つの利用モード: 自分の API Key、または任意のプリペイド Managed Credits を利用可能。
- 個人設定: AI プロバイダー、モデル、文体、カスタム prompt、BYOK モード用の API Key を保存。
- 多言語インターフェース: Chrome/システム言語に自動追従、または設定で手動切り替え。
- フィードバック / 要望: 問題報告、感想、機能リクエストを拡張機能から送信。
- 便利なサイドバー: 作業中のページを離れずに翻訳結果を確認。
- 簡単なオン/オフ: 必要なときだけ有効化できます。

English Writing Helper を使う理由

言いたい内容はあるけれど、より自然で読みやすい英語にしたい人向けです。文章を選択し、翻訳し、文体を調整し、結果をコピーして、そのまま作業を続けられます。

プライバシーについて

API Key は Chrome 拡張機能の保存領域に保存され、拡張機能の更新後も保持されます。現在の URL、閲覧履歴、API Key、原文、翻訳文が開発者へ自動送信されることはありません。

匿名利用統計は初回に明確な選択肢として表示されます。含まれるのは翻訳回数、文字数、プロバイダー、モデル、文体、拡張機能バージョンなどの集計情報のみです。匿名統計が有効な間、Managed Credits の今後の購入には 5% の bonus credits が付きます。

Managed Credits のプライバシー: Managed Credits モードでは、選択したテキストが翻訳と prepaid credits の差し引きのために English Writing Helper バックエンドと AI プロバイダーへ送信されます。利用状況 Dashboard に原文や翻訳文は保存されません。
