# English - en

## Title
English Writing Helper

## Summary
Translate and polish writing into English with formal, casual, or custom writing styles.

## Description
English Writing Helper helps non-native English writers translate and polish text into natural English across the web. Use your own Gemini or OpenAI API key, choose the tone that fits your context, and keep the helper available while writing emails, posts, messages, notes, and more.

Key features

- Real-time translation: Translate non-English text into English as you type or from selected text.
- Style options: Choose between formal writing and casual conversational English.
- Custom translation instructions: Define your own prompt for specific tones, formats, audiences, or writing rules.
- Gemini/OpenAI support: Use Google Gemini or OpenAI for high-quality English writing suggestions.
- Personal settings: Save your own API key, preferred provider, model, writing style, and custom prompt.
- Localized interface: Follow your Chrome/system language automatically, or switch languages manually in settings.
- Feedback and wishlist: Send bug reports, product feedback, or feature wishes directly from the extension.
- Flexible display: Read suggestions in a convenient sidebar while you keep working on the page.
- Easy control: Toggle the extension on or off whenever you need.

Why install English Writing Helper?

It is designed for people who already know what they want to say, but want clearer, more natural English. It keeps the workflow simple: select text, translate, adjust the style, copy the result, and keep writing.

Privacy notes

Your API key is stored in Chrome extension storage. The extension does not automatically send your page URL, browsing history, API key, source text, or translated text to the developer. Optional anonymous usage statistics only include aggregate counts such as translation attempts, character counts, selected provider, writing style, and extension version.
