# English - en

## Title
English Writing Helper

## Summary
Translate and polish writing into English with formal, casual, or custom writing styles.

## Description
English Writing Helper helps non-native English writers translate and polish text into natural English across the web. Use your own Gemini or OpenAI API key, or use optional Managed Credits if you do not want to set up an API key. Choose the tone that fits your context, and keep the helper available while writing emails, posts, messages, notes, and more.

Key features

- Real-time translation: Translate non-English text into English as you type or from selected text.
- Style options: Choose between formal writing and casual conversational English.
- Custom translation instructions: Define your own prompt for specific tones, formats, audiences, or writing rules.
- Gemini/OpenAI support: Use Google Gemini or OpenAI for high-quality English writing suggestions.
- Two usage modes: Use your own API key, or use optional prepaid Managed Credits.
- Personal settings: Save your preferred provider, model, writing style, custom prompt, and API key when using BYOK mode.
- Localized interface: Follow your Chrome/system language automatically, or switch languages manually in settings.
- Feedback and wishlist: Send bug reports, product feedback, or feature wishes directly from the extension.
- Flexible display: Read suggestions in a convenient sidebar while you keep working on the page.
- Easy control: Toggle the extension on or off whenever you need.

Why install English Writing Helper?

It is designed for people who already know what they want to say, but want clearer, more natural English. It keeps the workflow simple: select text, translate, adjust the style, copy the result, and keep writing.

Privacy notes

Your API key is stored in Chrome extension storage and is preserved across extension updates. The extension does not automatically send your page URL, browsing history, API key, source text, or translated text to the developer.

Optional anonymous usage statistics are requested with a clear first-run choice. They only include aggregate counts such as translation attempts, character counts, selected provider, model, writing style, and extension version. Managed Credits purchases receive 5% bonus credits while anonymous statistics are enabled.

Managed Credits privacy note: in Managed Credits mode, selected text is sent to the English Writing Helper backend and AI provider only to complete the translation and deduct prepaid credits. Source text and translations are not stored in the usage dashboard.
